# ბილდინგი

საჭიროა: [pnpm](https://pnpm.io/), [web-ext](https://github.com/mozilla/web-ext)

ექსტენშენის ზიპების გაკეთებისთვის:

```
pnpm package
```

მხოლოდ ბილდისთვის:

```
pnpm build
```

ნახეთ [package.json](package.json) მეტი სკრიპტისთვის
